This folder contains common reusable functionality used by other scripts.
To include all functionality use 'all.php', it includes all other files.

Classes:
Each class has a camel-case name and is stored in a separate file named
after the class, e.g. the class 'CommandLineParser' is stored in the 
file 'CommandLineParser.php'

Functions:
Functions are stored in the 'functions.php' file.
